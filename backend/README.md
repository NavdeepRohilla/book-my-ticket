# Backend

## Run

```bash
npm start
```

## Scripts

- `npm start` starts the backend server
- `npm run dev` starts the backend with Node watch mode

## Main Endpoints

- `POST /auth/register`
- `POST /auth/login`
- `GET /movies`
- `GET /seats`
- `GET /movies/:movieId/seats`
- `POST /bookings`
- `GET /bookings/me`
- `PUT /:id/:name`
